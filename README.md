# Swipy Nav

Run the app locally

```sh
npm install
npm start
```

Or see the [online demo](https://fibo.github.io/swipy-nav)

## Deploy

Demo is deployed on GitHub Pages via [this GitHub Action](./.github/workflows//gh-pages.yml).
To trigger it just push on `gh-pages` branch.
